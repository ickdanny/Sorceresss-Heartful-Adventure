---------------------------------------------
Controls

Menu Navigation
z = select
x = back
arrow keys = navigation

Gameplay
shift = focus
z = attack
x = bomb
arrow keys = movement
esc = pause

If game window does not fit your screen imput scaling ratio into frame_size_multi.
If you want to change the synth put the desired name into default_synth.
